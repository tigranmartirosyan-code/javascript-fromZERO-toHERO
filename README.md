# JavaScript from Zero to Hero

A comprehensive JavaScript learning curriculum from basics to React.

## Installation

```bash
npm install
```

## Project Structure

```
javascript-fromZERO-toHERO/
├── phase1-basics/       # JavaScript fundamentals
├── phase2-dom/          # DOM manipulation & Browser APIs
├── phase3-modern-js/    # ES6+ features
├── phase4-typescript/   # TypeScript
├── phase5-react/        # React with TypeScript
├── package.json
└── tsconfig.json
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm start` | Start live-server for HTML/JS files |
| `npm run dev` | Start live-server on port 3000 |
| `npm run ts <file>` | Run TypeScript file with ts-node |
| `npm run ts:watch <file>` | Run TypeScript file with hot reload |
| `npm run build:ts` | Compile TypeScript to JavaScript |

## Phases

### Phase 1: JavaScript Basics
Variables, data types, loops, functions, arrays, objects

### Phase 2: DOM & Browser APIs
DOM manipulation, events, timers, fetch, localStorage

### Phase 3: Modern JS (ES6+)
Template literals, destructuring, spread, arrow functions, classes, Map/Set

### Phase 4: TypeScript
Types, interfaces, generics, utility types

### Phase 5: React
Components, hooks, routing, state management

## Usage Examples

```bash
# Run Phase 2 DOM tasks
npm start
# Then open http://localhost:8080/phase2-dom

# Run TypeScript file
npm run ts phase4-typescript/example.ts
```
